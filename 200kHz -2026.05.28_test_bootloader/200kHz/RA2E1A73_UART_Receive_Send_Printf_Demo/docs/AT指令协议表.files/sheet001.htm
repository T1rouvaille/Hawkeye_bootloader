一、光功率校准 (AT+POWCAL)
功能：PC上位机读取光功率计 → 发AT+POWCAL → MCU自动调整参考电压 → 循环至目标光功率
序号 发送命令 正常响应 错误响应 说明
1. 上报光功率读数
1.1 AT+POWCAL=H,150 +POWCAL:H,ADJ,1550mV +POWCAL:ERR,H,SAT,3300mV 上报H通道光功率1.50mW(值x100)
ADJ=已调整需继续 / DONE=到位
1.2 AT+POWCAL=V1,100 +POWCAL:V1,ADJ,1200mV +POWCAL:ERR,V1,LASER_OFF,0mV 上报V1通道光功率1.00mW
注意: 需先确保激光已开启
1.3 AT+POWCAL=V2,85 +POWCAL:V2,DONE,980mV +POWCAL:ERR,V2,PD_LOW,30mV 上报V2通道光功率0.85mW
DONE=误差在±0.05mW内, 校准完成
2. 状态与结果
2.1 AT+POWCAL=STATUS +POWCAL:STATUS,H,ADJ,1550mV,target=200 — 查询全部三路校准状态
+POWCAL:STATUS,V1,IDLE,0mV,target=100
+POWCAL:STATUS,V2,DONE,980mV,target=100
2.2 AT+POWCAL=SAVE +POWCAL:SAVE,OK +POWCAL:SAVE,FAIL 保存校准结果到Flash
+RESP:LOCKED 成功后自动锁定
3. 停止校准
3.1 AT+POWCAL=STOP,H +POWCAL:STOP,H +POWCAL:ERR,UNKNOWN CHANNEL 停止H通道校准, 重置为IDLE
3.2 AT+POWCAL=STOP +POWCAL:STOP,ALL — 停止全部三路校准
4. 状态码含义
状态码 响应前缀 含义 后续操作
ADJ +POWCAL 已调整参考电压, 等待下次测量 继续读功率计并发AT+POWCAL
DONE +POWCAL 到位, 误差在±0.05mW内 可发AT+POWCAL=SAVE保存
SAT +POWCAL:ERR 饱和, 已达电压上限3300mV或下限0mV 检查激光/PD是否正常
LASER_OFF +POWCAL:ERR 激光未打开, 拒绝调节 先开启对应通道激光
PD_LOW +POWCAL:ERR PD初始值低于50mV, 拒绝调节 检查PD传感器连接
NO_RESP +POWCAL:ERR 升压但光功率未响应, 拒绝调节 检查激光管/光路
二、参考电压配置
功能：手动设置三路激光的PD参考电压(V2/H/V1) 及锁定/保存
序号 发送命令 正常响应 错误响应 说明
1. 解锁与锁定
1.1 AT+UNLOCKCFG=123456 +RESP:UNLOCKED +RESP:PASSWORD ERROR, PLEASE RETRY 输入密码解锁配置模式
1.2 AT+LOCKSTATE? +RESP:LOCKED +RESP:UNLOCKED 查询当前锁定状态
2. 设置参考电压（需先解锁）
2.1 AT+SET_FRONT=1500 +RESP:OK +RESP:PLEASE UNLOCK FIRST 设置V2(FRONT)参考电压1500mV
+RESP:ERR 范围: 0~3300mV
2.2 AT+SET_SIDE=1200 +RESP:OK +RESP:PLEASE UNLOCK FIRST 设置H(SIDE)参考电压1200mV
+RESP:ERR 范围: 0~3300mV
2.3 AT+SET_HORIZ=1300 +RESP:OK +RESP:PLEASE UNLOCK FIRST 设置V1(HORIZ)参考电压1300mV
+RESP:ERR 范围: 0~3300mV
3. 查询参考电压
3.1 AT+GET_FRONT? +RESP:FRONT=1500mV — 查询V2(FRONT)当前值
3.2 AT+GET_SIDE? +RESP:SIDE=1200mV — 查询H(SIDE)当前值
3.3 AT+GET_HORIZ? +RESP:HORIZ=1300mV — 查询V1(HORIZ)当前值
3.4 AT+GET_ALL RESP:FRONT=1500mV — 一次查询三路参考电压
RESP:SIDE=1200mV
RESP:HORIZ=1300mV
4. 保存与重置
4.1 AT+SAVE +RESP:SAVE OK +RESP:PARAMETER NOT READY (NOT LOCKED) 保存参考电压到Flash(需锁定状态)
三路全部设置后自动锁定
4.2 AT+RESET +RESP:RESET SUCCESS, FLASH ERASED +RESP:PLEASE UNLOCK FIRST 清零RAM中的参考电压并擦除Flash
+RESP:RESET FAILED 需先解锁

八、激光串口控制 (AT+LASER)
功能：串口控制单路激光开关, 支持逐路校准流程(H → V1 → V2)
序号 发送命令 正常响应 错误响应 说明
1. 单路开关控制
1.1 AT+LASER=H,ON +LASER:H,ON +LASER:ERR,BUSY 开启H(SIDE)通道激光
+LASER:ERR,FORMAT 首路开启走MOS时序(设PWM→等10ms→EN1_ON)
1.2 AT+LASER=H,OFF +LASER:H,OFF +LASER:ERR,BUSY 关闭H(SIDE)通道激光
+LASER:ERR,FORMAT 末路关闭走MOS时序(EN1_OFF→等10ms→关PWM)
1.3 AT+LASER=V1,ON +LASER:V1,ON +LASER:ERR,BUSY 开启V1(HORIZ)通道激光
1.4 AT+LASER=V1,OFF +LASER:V1,OFF +LASER:ERR,BUSY 关闭V1(HORIZ)通道激光
1.5 AT+LASER=V2,ON +LASER:V2,ON +LASER:ERR,BUSY 开启V2(FRONT)通道激光
1.6 AT+LASER=V2,OFF +LASER:V2,OFF +LASER:ERR,BUSY 关闭V2(FRONT)通道激光
2. 全开/全关 (原子操作)
2.1 AT+LASER=ALL,ON +LASER:ALL,ON +LASER:ERR,BUSY 三路同时开启(原子: 先设gLaserOn再统一配置PWM+MOS)
2.2 AT+LASER=ALL,OFF +LASER:ALL,OFF +LASER:ERR,BUSY 三路同时关闭(原子: EN1_OFF→10ms→PWM全关)
3. 错误码说明
BUSY +LASER:ERR,BUSY MOS状态机正忙(EN1 10ms延时中) 等~20ms后重试
FORMAT +LASER:ERR,FORMAT: ... 命令格式错误, 缺少逗号或值不正确 格式: AT+LASER=<H|V1|V2|ALL>,<ON|OFF>
4. MOS状态机行为
场景 动作 时序
首路开启(之前全关) 设PWM载波 → 等10ms → EN1_ON 所有载波信号一起使能
追加开启(已有路亮) 直接设该路 PWM=60 无延时(EN1已使能)
非末路关闭 设该路 PWM=20(idle) 无延时(其他路仍用EN1)
最后一路关闭 EN1_OFF → 等10ms → PWM全0 干净关断时序
5. 逐路校准工作流程 (PC端)
AT+LASER=ALL,OFF // 确保干净初始状态
AT+LASER=H,ON // 只开H
AT+POWCAL=H,200 // 校准H (循环直到DONE)
AT+LASER=H,OFF
AT+LASER=V1,ON // 只开V1
AT+POWCAL=V1,100 // 校准V1
AT+LASER=V1,OFF
AT+LASER=V2,ON // 只开V2
AT+POWCAL=V2,100 // 校准V2
AT+POWCAL=SAVE // 三路保存到Flash
AT+LASER=V2,OFF
六、通道映射与校准参数
UART配置：SCI0, 115200bps, 8N1, RTS流控
参数 H (SIDE) V1 (HORIZ) V2 (FRONT) 说明
通道映射
通道枚举 POWCAL_CH_H = 0 POWCAL_CH_V1 = 1 POWCAL_CH_V2 = 2 power_calibration.h
AT指令通道名 H V1 V2 命令中使用的名称
gLaserOn 索引 [0] [1] [2] 激光开关状态
g_pd_avg 索引 [0] = pd_avg1 [1] = pd_avg2 [2] = pd_avg PD均值(mV)
reference_voltage 变量 reference_voltage1 reference_voltage2 reference_voltage 参考电压变量
校准目标参数 (hawkeye_config.h)
目标光功率(x100) 200 (2.00mW) 100 (1.00mW) 100 (1.00mW) POWCAL_TARGET_H/V1/V2
步进阈值(x100, 0.01mW)
大步进阈值 |误差| > 50 (0.50mW) → 步进 200mV POWCAL_THRESH_LARGE
中步进阈值 |误差| > 20 (0.20mW) → 步进 50mV POWCAL_THRESH_MED
小步进阈值 |误差| > 5 (0.05mW) → 步进 10mV POWCAL_THRESH_SMALL
到位条件 |误差| ≤ 5 (0.05mW) → DONE 　
保护参数
电压上限 3300mV POWCAL_VOLT_MAX_MV
电压下限 0mV POWCAL_VOLT_MIN_MV
PD最低阈值 50mV (低于此值禁止校准) POWCAL_PD_MIN_MV
七、光功率校准 — PC侧交互流程
以H通道为例 (目标2.00mW)，从初始到完成的完整交互序列：
轮次 PC 发送 MCU 响应 说明
初始状态: H通道激光已开启, PD=1500mV, 光功率计读数=0.50mW(偏低)
1 AT+POWCAL=H,50 +POWCAL:H,ADJ,1700mV 功率50(0.50mW) < 目标200(2.00mW)
误差150 > 大步进阈值50 → +200mV
2 AT+POWCAL=H,110 +POWCAL:H,ADJ,1900mV 功率110(1.10mW), 误差90 > 50 → +200mV
3 AT+POWCAL=H,165 +POWCAL:H,ADJ,1950mV 功率165, 误差35 > 20 → +50mV(中步进)
4 AT+POWCAL=H,190 +POWCAL:H,ADJ,1960mV 功率190, 误差10 > 5 → +10mV(小步进)
5 AT+POWCAL=H,198 +POWCAL:H,DONE,1960mV 功率198, 误差2 ≤ 5 → 到位!
校准完成, 保存结果
6 AT+POWCAL=SAVE +POWCAL:SAVE,OK 保存到Flash, 自动锁定
